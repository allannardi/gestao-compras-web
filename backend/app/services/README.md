# Serviços de negócio

Os serviços validados do projeto Streamlit serão migrados gradualmente para esta pasta.

Ordem recomendada:

1. funções puras de interpretação da NFC-e;
2. consolidação de itens;
3. normalização de produtos;
4. classificação automática;
5. persistência no PostgreSQL.

Não importar `streamlit` dentro desta camada.
