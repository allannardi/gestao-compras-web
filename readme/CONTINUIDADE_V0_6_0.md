# Continuidade — Gestão de Compras Web v0.6.0

## Base validada anterior

**v0.5.2 — Dashboard, histórico e seletor de mês compatível com iPhone.**

## Escopo desta versão

A v0.6.0 transforma a estrutura de Famílias em uma operação compartilhada real:

- configurações pessoais e familiares;
- membros;
- convites;
- papéis;
- troca entre famílias;
- contador de registros no histórico de preços.

## Convites

O administrador cadastra o e-mail. A pessoa entra com o mesmo endereço e aceita o convite em **Ajustes**. O envio automático de e-mail não faz parte desta versão.

## Confirmação de e-mail

O app está preparado para acesso imediato após o cadastro, mas a opção `Confirm email` precisa ser desativada manualmente no Supabase Auth.
