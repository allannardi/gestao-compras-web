# Changelog — v0.6.4

## Categorias

- nova área de cadastros mobile;
- lista de categorias ativas e desativadas;
- indicadores de categorias e produtos classificados;
- criação de categoria personalizada;
- alteração do nome de categorias personalizadas;
- desativação com escolha obrigatória da categoria de destino;
- transferência segura dos produtos antes da desativação;
- reativação de categorias personalizadas;
- categorias do sistema permanecem protegidas.

## Supermercados

- lista de supermercados da família;
- quantidade de compras, total registrado e última compra;
- correção do nome exibido;
- união segura de cadastros duplicados quando o CNPJ permite preservar futuras leituras;
- transferência de compras e histórico de preços para o cadastro principal;
- cadastros antigos unidos deixam de aparecer nos filtros.

## Navegação

- novo botão `Cadastros` ao lado de `Ajustes`;
- interface em cards, sem modal pesado;
- layout preparado para iPhone.

## Segurança

- somente administradores alteram, desativam ou unem cadastros;
- membros continuam podendo consultar as informações;
- todas as operações permanecem isoladas por `familia_id`.
