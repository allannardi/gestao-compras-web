# Gestão de Compras Web

Aplicação mobile/PWA para registrar e acompanhar compras domésticas por NFC-e.

## Patch atual

**v1.2.1 — Hotfix da galeria de fotos**

O hotfix separa a seleção de uma foto já existente da abertura da câmera. O botão
**Usar uma foto já tirada** passa a abrir a galeria/selecionador de arquivos,
enquanto o botão de leitura pela câmera mantém o comportamento atual.
